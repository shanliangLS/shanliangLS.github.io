处理文件路径 windows 与 linux 兼容问题

window 和 linux 分隔符其实就是 \ 和 / 的区别，其实让文件路径根据服务器的系统的不同而不同就可以了，而 java 已经给我们提供方法了 System.getProperty("file.separator") 能根据系统的不同获取文件路径的分隔符

在拼接文件路径时尽量使用File.separator，避免平台路径不兼容造成的错误
